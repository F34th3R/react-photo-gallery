#🤳🏻 Photo Gallery

🚀 React App using Framer motion and Firebase ✨!

![Drag Racing](./doc/app_screenshot.png)


## Step 00 | Install all dependencies

```bash
$ npm install
```

## Step 01 | Config firebase

Go into `src/firebase/config.ts` and add your `firebaseConfig`.

```javascript
var firebaseConfig = {
 apiKey: "",
 authDomain: "",
 databaseURL: "",
 projectId: "",
 storageBucket: "",
 messagingSenderId: "",
 appId: ""
}
```

## Step 02 | Run!

```bash
$ npm start
```

## Step 03 | Build

```bash
$ npm run build
```
